# TimiCaller31
### * 作品理念與開發過程
初始發想是具有鬧鐘功能的代辦事項清單，後加入事項公佈欄、位置歷史紀錄、發票兌獎等功能。<br/>
>過程中學習到:
>>。api的使用<br/>
>>。android版本的差異<br/>
>>。資料庫管理應用<br/>
